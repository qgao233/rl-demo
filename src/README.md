# Q-Learning 迷宫求解

这是一个使用Q-Learning算法解决4x4迷宫问题的Python实现。

## 迷宫布局

```
S . . .
. # . #
. . . .
# . . G
```

- `S`: 起点 (0,0)
- `G`: 终点 (3,3)
- `#`: 障碍物
- `.`: 可通行路径

## 文件说明

- `maze_env.py`: 迷宫环境实现
- `q_learning.py`: Q-Learning算法实现
- `train.py`: 训练和测试脚本

## Q-Learning算法

### 更新公式

```
Q(S,A) ← Q(S,A) + α[R + γ max Q(S',a') - Q(S,A)]
                           a'
```

### 参数设置

- **学习率 (α)**: 0.1
- **折扣因子 (γ)**: 0.9
- **探索概率 (ε)**: 0.1

### 策略

- **行动策略**: ε-greedy（10%随机探索，90%选择最优动作）
- **评估策略**: 贪婪策略（选择最大Q值）
- **类型**: 异策略（off-policy）

## 使用方法

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行训练

```bash
cd src
python train.py
```

### 3. 输出结果

程序会输出：
- 训练过程中的进度信息
- 训练完成后的Q表（部分）
- 测试阶段的最优路径
- 训练结果可视化图表（保存为`training_results.png`）

## 奖励设置

- 到达终点: +100
- 撞墙/障碍物: -1
- 正常移动: -0.1（鼓励尽快到达终点）

## 预期结果

经过500轮训练后，智能体应该能够：
- 从起点(0,0)出发
- 避开障碍物
- 找到最短路径到达终点(3,3)
- 成功率接近100%
